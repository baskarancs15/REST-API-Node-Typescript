# Development

You should have installed at least node version 14.0.

Clone the repository:
https://github.com/baskarancs15/REST-API-Node-Typescript.git

Install the dependencies within the project directory:
npm install

To run Server:
npm start

To run Test:
npm test

To run build:
npm run build
